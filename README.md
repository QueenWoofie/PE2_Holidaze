# Holidaze, Glaze your holiday with Holidaze

A modern venue booking site with a bit of a fairytale vibe in connection with project exam 2 for Noroff.

## üöÄ Features

- **Browse Venues**: View and search through available venues
- **User Authentication**: Register and login with Noroff API authentication
- **Book Venues**: Make bookings for selected dates and number of guests
- **Cancel Bookings**: Cancel booked bookings
- **Manage Venues**: Manage your bookings and view booking history
- **Profiles**: User profiles with profile image
- **Venue Manager**: Special role for users who want to list their own venues

## üîß Setup & Installation

1. Clone the repository:

```bash
git clone https://github.com/QueenWoofie/PE2_Holidaze.git
```

2. Open the project in your editor.

3. Install dependencies:

```bash
npm install
```

4. Create environment file:

Create a .env file in the root of the project and add:
VITE_NOROFF_API_BASE_URL=https://v2.api.noroff.dev
VITE_NOROFF_API_KEY=your_api_key_here

5. Start the development server:

```bash
npm run dev
```
